Principles of Programming Languages II

601.427/627   Fall 2019   Prof. Scott Smith

Course Summary

    • This course is a an advanced study of important modern topics in the principles of programming languages. It builds on the material of 601.426/626, Principles of Programming Languages.
    • Topics covered include operational semantics, type systems, program analysis, program logics, and program verification.
    • Programming in the course will be in OCaml, we will assume students are familiar with OCaml or canb pick it up on their own.
    • This course is recommended for students that already have background in programming language principles, e.g. by taking 601.426/626. If you have this background email the professor to obtain permission to add the class.
